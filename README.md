## 🚀 How to run project

### 1. Install dependencies

```bash
npm install
````

---

### 2. Start development server

```bash
npm run dev
```

App will be available at:

```
http://localhost:5173
```

---

### 3. Build production version

```bash
npm run build
```

---

### 4. Preview production build

```bash
npm run preview
```

---

### 5. Run lint

```bash
npm run lint
```
